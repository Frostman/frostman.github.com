Практическое задание (на занятии) #1 / Домашнее задание

Необходимо дорешать задачи предыдущего домашнего задания. Так же к ним прибавляется задача 370 с того же сайта (acm.sgu.ru/univer, которую так же необходимр еешить с использованием компилятора Java 6 + Checkstyle (JAVA-X), но еще одним обязательно условием явлется использование принципов ООП, т.е. вы должны создать некоторые объектную модель для решения этой задачи. Только решения отправленные с использованием этого компилятора и использующие принципы ООП будут засчитываться.
